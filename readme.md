博客后端使用Laravel Lumen搭建，配合ElementUi 做的SPA单页应用，本人博客地址(https://6pixels.com)


## 安装配置

###安装方法
```git
    git clone https://github.com/RoseEnd/blog.git
```
### 执行数据库迁移
```git
    cd blog
    php artisan migrate
```
### 初始化后台管理员账号
```
    php arisan admin
```
### 安装前端依赖
```git
    cd resources
    npm install
```

安装完成，访问(http://localhost/admin) 登陆体验
